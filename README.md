## Project Overview

This repository contains my Personal Web Profile developed for the CMSC 207 Mini-Project.

The objective of this project is to apply foundational web development principles including:

- Semantic HTML structure  
- External CSS styling  
- Basic JavaScript interactivity  
- Responsive design for desktop and mobile  

The site presents a formal professional profile including background, education, interests, and contact information.

---

## Project Objectives Achieved

- Developed a structured and semantic HTML page  
- Applied external CSS for layout, colors, and typography  
- Implemented JavaScript interactivity (Fun Fact button)  
- Designed a responsive layout using media queries  
- Practiced clean code organization and debugging  

---

## Technologies Used

- **HTML5** – Semantic structure (`header`, `section`, `footer`, `article`, etc.)
- **CSS3** – External stylesheet with white and green theme
- **JavaScript** – Dynamic content generation for fun fact interaction

## Reflection

A major challenge faced during the creation of this mini-project was finding a balance between professional complexity and the educational goal of basic web development techniques. With my background in digital transformation and enterprise systems, my first inclination was to use more intricate layouts and sophisticated design models. However, the purpose of the project called for a careful focus on semantic HTML, external CSS styling, and lightweight JavaScript interactivity. This required a thorough examination of content scope and architectural simplicity to meet project specifications while ensuring a polished and professional look.  

I encountered difficulties when designing the Licenses and Certifications section, which was intended to utilize a two-column grid layout that remained visually balanced and responsive across different screen sizes. Initial tries led to misaligned cards and inconsistent spacing, especially on smaller screens. These problems called for repeated adjustments of CSS grid properties and the use of media queries to maintain structural integrity and readability. Through this experience, I gained a better understanding of responsive design principles and the significance of visual hierarchy in user interface development. Ultimately, this project underscored the essential role of semantic structure, modular styling, and gradual enhancement in creating scalable digital experiences, demonstrating that successful digital platforms are built on strong technical fundamentals.
